# MaterialDesignColor

> 谷歌官方的MaterialDesign颜色代码

### 使用

```compile 'com.zzhoujay.materialcolors:materialcolors:1.0.0'```

在代码中使用诸如```R.color.material_blue_500```来引用对应的颜色值，其中`material`为前缀，`blue`为颜色类别，`500`为颜色深度
